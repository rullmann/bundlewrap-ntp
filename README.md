# bundlewrap-ntp

`bundlewrap-ntp` does install and configure ntp to run as a service and use ntp servers located in Europe.
It does bind to 127.0.0.1 only.

## Compatibility

This bundle has been tested on the following systems:

| OS          | `[x]` |
| ----------- | ----- |
| CentOS 7    | `[x]` |
| Fedora 24   | `[x]` |
| RHEL 7      | `[x]` |
| Fedberry 23 | `[ ]` |
